## Usage

1. Adjust maze size and obstacle density using the sliders
2. Click "Generate Maze" to create a new random maze
3. Choose an algorithm:
   - **Run BFS** - Guarantees shortest path
   - **Run DFS** - Memory efficient exploration
   - **Compare Both** - Run both algorithms sequentially
4. Watch the animated visualization and compare statistics

## Local Development
